### 目录结构

```
├── src
│   ├── api // 接口请求
│   ├── assets // 全局静态资源
│   ├── components // 通用业务组件
│   ├── constants // 常量
│   ├── directives // 自定义指令
│   ├── libs // 通用中前台物料组件
│   ├── router // 路由
│   │   └── modules // 路由模块
│   ├── store // 全局状态管理
│   │   └── modules // 状态管理模块
│   ├── styles // 全局样式
│   ├── utils // 工具类
│   ├── vendor // 第三方库
│   └── views // 视图
│       └── layout // 布局
│           ├── components // 布局组件 
```

### 运行

```shell
npm install
npm run dev
```
### 打包

```
npm run build
```
